This project is a standalone web application and **does not require Node.js or any build steps.**

### Local Development
Simply open `index.html` in your web browser.

### Hosting on GitHub
1. Create a new repository on GitHub.
2. Upload all files from this folder to the repository.
3. Go to **Settings** > **Pages**.
4. Under **Build and deployment**, select the `main` branch and `/root` folder.
5. Click **Save**. Your site will be live in a few minutes!
